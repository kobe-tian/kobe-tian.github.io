前言

什么是Python

         记得2016年8月的时候，我第一次接触python。我就对它很好奇，比如: 这个print好神奇、wa if-else 居然还可以玩 等等现在看起来很弱智的问题。当然我并不像很多科班出生的同学那样，对计算机基础，编程语言，算法知识有比较系统性的学习和了解。所以，如果你看到这篇文章，嗯嗯，我想我们是同一战线的。
         好吧，言归正传，什么是python？我们先来看看官方的解释: Python是一种多范式，通用，解释的高级编程语言。是不是有点懵逼，别着急后面还有一段话: Python允许程序员使用不同的编程风格来创建简单或复杂的程序，获得更快的结果并编写代码，就像用人类语言说话一样。 一句话概括，Python学习起来很简单，写Python就和你自己说话一样。作为一个Pythoner 这一点我非常赞同。当然Python还有很多很多的优点哈，这里就不一一举例了。

Python可以做什么

以我短暂的工作经验来看, 大概可以分为以下几类

  • web开发: Python有很多优秀的框架 Django、Flask、Tornado
  • 网络爬虫: aiohttp、Scrapy
  • 数据分析: pandas 、numpy
  • 自动化运维: 主要还是用到python自身的一些属性
  • 人工智能: 机器学习-Scikit-learn、深度学习-TensorFlow
